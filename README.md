Problema:

Pedidos de uma cafeteria:
Você tem uma lista de pedidos feitos para uma cafeteria, descubra quantos pedidos de café foram feitos no dia.

Regras:

Entrada: Lista de pedidos
Saída: Quantidade de pedidos de "café" durante o dia

Testes:
TC 1 - Ao menos 1 café na lista
TC 2 - Não ter nenhum café na lista
TC 3 - Ter 2 ou mais
TC 4 - Testar incluindo café e café com leite, e deve contabilizar apenas um café
